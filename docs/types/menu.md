title: CutieMenu QML Type
---

CutieMenu is a component used to provide hamburger menus and context menus styled to fit Cutie UI visual language.

Inherits: QtQuick.Controls 2.15 Menu

## Detailed Description

This menu type defaults to be a modal menu. Usually, it is used to provide a value to the menu properties of [CutieListView](listview) and [CutieListItem](listitem). As for the menu items inside, it is highly recommended to use [CutieMenuItem](menuitem).
